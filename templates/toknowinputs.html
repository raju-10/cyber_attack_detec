How to Find Input Values of Your Device

Perform the following steps:

  1. Install Wireshark

    Wireshark is a powerful network protocol analyzer that can provide most of the information you need. You can download and install the latest version of Wireshark from their official website (https://www.wireshark.org/download.html).

  2. Start Wireshark and capture traffic

    After installing Wireshark, open the application and start capturing packets by selecting the appropriate network interface (usually the Wi-Fi or Ethernet interface). You can choose to capture traffic for a specific duration or continuously.

  3. Analyze the captured traffic

    Once you have captured the desired traffic, you can use the following methods to find the properties you need:

    • Mean flow (mean)

      In Wireshark, go to "Statistics" > "Conversations" and select the "TCP" tab. You can see the duration of each flow, and you can export the list as CSV file using the copy option to calculate the average duration manually. To calculate Mean of 'duration' you can use following ways:

      1. Create a python notebook → Upload CSV file into it → Now run the following code to get the 'mean of duration'.

        import pandas as pd

# Read the CSV file into a pandas DataFrame
df = pd.read_csv('myfile.csv')

# Calculate the average of the "duration" column
average_duration = df['duration'].mean()

print("Average duration:", average_duration)

      2. You can manually calculate the average of the values of the column 'duration' using the formula:

        mean = Sum of all durations/no of durations = d1+d2+d3+...,/n

    • Destination Port (Dport)

      In the main Wireshark window, you can see the destination port (port B) number for each packet.

    • Source Packets (Spkts)

      Go to "Statistics" > "Conversations" to see the Packets sent from source to destination (A→B).

    • Destination Packets (Dpkts)

      Go to "Statistics" > "Conversations" to see the Packets sent from destination to source (B→A).

    • Total Packets (Tpkts)

      Sum of 'Spkts and Dpkts' = Source Packets + Destination Packets.

    • Destination Bytes (Dbytes)

      Go to "Statistics" > "Conversations" to see the number of bytes sent from destination to source (B→A).

    • Source Bytes (Sbytes)

      Go to "Statistics" > "Conversations" to see the number of bytes sent from source to destination (A→B).

    • Total Bytes (TBytes)

      Sum of 'Dpkts and Spkts' = Destination Packets + Source Packets.

    • Source Load (Sload)

      Sload = (Source Bytes * 8) / Capture Duration (in seconds).

    • Destination Load (Dload)

      Dload = (Destination Bytes * 8) / Capture Duration (in seconds).

    • Total Load (Tload)

      Tload = (Total Bytes * 8) / Capture Duration (in seconds).

    • Source Rate (Srate)

      Srate = Source Packets / Capture Duration (in seconds).

    • Destination Rate (Drate)

      Drate = Destination Packets / Capture Duration (in seconds).

    • Total Rate (Trate)

      Trate = Total Packets / Capture Duration (in seconds).

    • Source Loss (Sloss)

      Go to "Analyze" > "Expert Information" and look for the count of 'retransmissioned' packets under "Group" Category of TCP Protocol. If there is no 'rentransmitted group' then it implies that there are no retransmitted packets (Sloss = 0).

    • Destination Loss (Dloss)

      Go to "Analyze" > "Expert Information" and look for the count of 'Out of Order' packets under "Group" Category of TCP Protocol. If there is no 'Out of Order group' then it implies that there are no out of order transmissioned packets (Dloss = 0).

    • Total Loss (Tloss)

      Tloss = Source Loss + Destination Loss.

    • Total Percent Loss (Ploss)

      Ploss = (Total Loss / Total Packets) * 100.
